# SdraKb00

11 keys hot-swap macropad with rotary encoder.


## Features

- 11 keys
- Rotary encoder
- AtMega32U4 MCU
- Per-key RGB led for backlighting
- USB-C connector
- On PCB SPI header
- MCU reset button
- Switch mounting plate
- Power LED indicator
