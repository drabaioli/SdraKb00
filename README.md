# SdraKb00

11 keys custom mechanical keyboard with rotary encoder.

## Features

- 11 keys
- rotary encoder
- AtMega32U4 MCU
- Per-key RGB led for backlighting
- USB-C connector
- On PCB SPI header
- MCU reset button
